# Common Scripts

Shared utilities used across phases.

- `summarize_results.py` - statistical summaries for experiment outputs (Phase 1 and Phase 2)
- `audit_phase2_legacy_artifacts.py` - immutable sidecar audit for legacy Phase 2 v1 runs.
- `audit_parser_behavior.py` - parser-mode agreement/ambiguity audit from intervention prediction samples.
- `power_analysis.py` - preregistration-driven power planning report generator.
